# ASPNETCoreAngular
ASPNET Core Angular (VS2017 or csproj)

For VS2015 or project.json version, please see: https://github.com/Longfld/ASPNETCoreAngular4

after download:

>dotnet restore

>npm install

>npm start

>dotnet run


And do not forget to run "npm start" after modify your any typescript files
